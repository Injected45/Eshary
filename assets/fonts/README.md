# Font assets

## Almarai (primary, app + PDF)

Registered in `pubspec.yaml` under family `Almarai`. The app theme uses
this as its default `fontFamily`, and `lib/shared/pdf_export.dart` loads
Regular + Bold from this set.

- `Almarai-Light.ttf`     — weight 300
- `Almarai-Regular.ttf`   — weight 400 (default)
- `Almarai-Bold.ttf`      — weight 700
- `Almarai-ExtraBold.ttf` — weight 800

Source: https://fonts.google.com/specimen/Almarai
Mirror used: https://github.com/google/fonts/tree/main/ofl/almarai
